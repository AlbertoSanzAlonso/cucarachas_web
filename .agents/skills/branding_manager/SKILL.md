# Skill: Branding Manager (CECSA)

# Propósito
Garantizar que todos los desarrollos visuales y estilos sigan estrictamente las directrices del manual de branding de CECSA (2020).

# Reglas de Oro
1.  **Colores Estrictos**: No usar colores fuera del manual.
    -   Azul Corporativo: `#0080bb`
    -   Gris Secundario: `#3c3c3b` o Negro 90%.
    -   Fondo Primario: Blanco o Gris Muy Claro (#F9FAFB).
2.  **Tipografía**:
    -   Headings: **Montserrat**.
    -   Body: **Segoe UI** o Inter como fallback.
3.  **Aesthetics**: Premium, limpio y profesional. Inspiración en sitios de autoridad (como Anticimex) pero con los colores de Cecsa.

# Procedimiento de Cambio
-   Al modificar cualquier archivo CSS o componente UI, verifica que no se introduzcan colores "ad-hoc".
-   Mantener las variables CSS en `index.css` siempre actualizadas.
