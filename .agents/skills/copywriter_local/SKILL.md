# Skill: Copywriter Local (Barcelona)

# Propósito
Crear textos potentes, discretos y con autoridad que resuenen con el público de Barcelona y su área metropolitana.

# Reglas de Tono
1.  **Filosofía CECSA**: Usar el enfoque "Aceptar para actuar con conciencia". Evitar el sensacionalismo del miedo a las plagas.
2.  **Referencia Geográfica**: Asegurar que los barrios (Eixample, Gràcia, etc.) y zonas (Vallès, Maresme) se mencionen con naturalidad para construir confianza local.
3.  **Autoridad Sanitaria**: Siempre mencionar el cumplimiento del ROESB (0246-CAT-SB) en textos de captación comercial.

# Vocabulario Preferido
-   "Control de plagas preventivo"
-   "Restauración del equilibrio"
-   "Eficiencia discreta"
-   "Certificados sanitarios en 24h"
