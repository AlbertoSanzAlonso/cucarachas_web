# CECSA Frontend Application

This directory contains the React-based frontend for the CECSA Control de Plagas platform.

## Quick Start

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build
```

## Tech Highlights

- **Vite** for ultra-fast HMR and bundling.
- **Tailwind CSS v4** for modern, utility-first styling.
- **Framer Motion** for premium UI transitions.
- **i18next** for multilingual support.

For more information on the project architecture and philosophy, see the [main README](../README.md).
